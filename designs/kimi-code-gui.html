设计原型

Kimi Code GUI

为 kimi-code CLI 构建的桌面客户端。采用 CodePilot 界面风格，Electron + React + Vite 技术栈，通过子进程方式接入 kimi-code。

Electron 主进程

管理窗口、IPC 通信、子进程生命周期。使用 Node.js child_process.spawn 启动 kimi-code CLI。

React 渲染进程

构建响应式 UI，处理消息列表、输入框、侧边栏、文件浏览器等交互组件。

kimi-code CLI

通过 -p --output-format stream-json 参数启动非交互模式，逐行解析 NDJSON 流式输出。
